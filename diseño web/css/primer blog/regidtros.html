Tipos de CSS selectors:
Para selecionar en CSS se debe de usar un selector que varia en tipos.
Universal(*)
De tipo(selecionamos la etiqueta que queremos modiicar)
De clases(Le damos a una etiqueta una clase y el CSS seleciona a todas las etiquetas de esa misma lcase para modificarlas. Antes de poner la clase en el selector se debe de poner un punto para indicar que es una clase)
Por ID(Le damos una ID a una etiqueta y luego para pner el nombre de la etiqueta en el selecro debemos de poner primero un "#")
Por atributos(seleccionamos ela tributo que queremos modificar y lograremos modificar a todas las etiquetas que contengan ese atributo. Para poder poner el atributo como selector demos colocar el atributo exacto dentro de corchetes)
De manera descendiente(sirve para seleccionar etiquetas dentro de otras etiquetas. para ponerlo como slector debemos poner de mayor a menor las cajas que lo contengan)
Por pseudo clase(se trata de las clases que le damos al selector. Ej: si utilizamos la ":hover", la modificacion solo actuara cuando pongamos el raaton encima del selector)

    9. medidas

    Para controlar las medidas de una etiqueta demos de utilizar la etiqueta "font-side". Para las medidas podemos utilizar medidas fijas(pixeles, cm, mm, etc.) y medidas relativas.
    Para las medidas relativas podemos usar "em" y "rem". En el caso de "em" este valdra aproximadamente 16px pero podra depender del navegador. Pero para darle una medida fija solo debemos de ponerle una medida a la caja contenedora.

    demas cosas

    con los atributos "padding" y "margin" podemos controlar los margenes de la pagina.
    Con el atributo "background" podemos controlar el condo de la etiqueta que seleccionemos y lo podemos sumar con otros atributos como "color", etc.
    Con el atributo "width" podemos controlar cuanto espacio ocupa de manera horizonta la etiqueta dentro de la pantalla. El espacio que ocupara se medira en porcentaje de la pantalla y para la unidad de medida debe de ser "vw".
    Con el atributo "height" podemos controlar el espacio que ocupa la etiqueta de manera vertical. y al igual que el atributo "width" se medira de manera porcentual pero con la unidad de medida "vh"

    Propiedadesd de texto

    La propiedad "font-family" que se utiliza para cambiar la tipografia de la fuente. Para implemeter tipografias externas solo se debe de copiar el link de l tipografia, colocarlo dentro de head de HTML dentro de una etiqueta "link" y poner el nombre de la tipografia dentro de "font-family".
    La propiedad "line-height" se utiliza para modificar el tamaño de la etiqueta dependiendo de su base, si el line-height es de dos este duplicara su tamaño desde el centro
    La propiedad "font-weight" nos sirve para modificar el grosor de la fuente.

    propiedades de cajas

    Las cajas se reigen por el box model, el box model es una regla de la propieda que es dirigida por cuatro atributos: el conten, el padding, el border y el margin. Para poder modificar las cajas de las etiquetas podemos usar la propiedad "padding", que la utilizamos para modificar la distancia que hay del texto a la caja. La caja se divide en cuatro laterales:"top, right, bottom y lef". se puede modificar cada lateral tansolo especificando en el atributo padding a cual se refiere.
    tambien para modificar la caja podemos utilizar el atributo display que nos sirve para modificar si la caja es en linea o si es en bloque.
    Con el atributo "text-aling" nos permite aliner el texto dentro de la caja de manera horizontal.
    con el atributo "margin" nos permite modificar el margen que hay entre las cajas. Al igual que el padding se puede manejar por top, right, bottom y left.
    Con el atribut "border-radius" podemos redondear los bordes de la caja, no del texto. Aun que tambien podemos usar el atributo "border en solitario oara agregar un borde.
    Tambien tenemos el atributo "box-shadow" que nos sirve para darle sombra a la caja. la primera caracteristica que le demos sera para controlr el eje Y, la segunda para el eje X, la tercera para el desenfoque y por ultimo el color. suecde los msimo con el atributo "text-shadow"
    Con el atributo "transform" podemos transformar la caja, si lo combinamos con la caracteristica rotate podemos rotar la caja, y esta caracteristica se rige por grados
