Etiquetas de Head

etiqueta "title"

La etiqueta "title" es utilizada para agregar un titulo a la pestaña de la pagina web.

etiquetas meta

la etiqueta meta sirve para controlar el comportamiento de la pagina web dependiendo de la caracteristica que le pongas el comportamiento resultara diferente. tambien se pueden combinar varios comportamientos en diferentes etiqueta de meta.
la etiqueta meta se puede escribir con las estiquetas "charset", "type", "keyword", "desciption", "author", "copyright" y "robots"
charset es una caracteristica que se utiliza para que la etiqueta meta puede leer ciertos caracteres que la pagina no puede.
type
keyword es una etiqueta que se utiliza para agregar palabras clave a la hora de la busqueda de la pagina web.
description es una etiquet utilizada para agregar una descripcion a la pagina web. esta aparece debajo del titulo a la hora de la busqueda.
author sirve para agregar un autor a la pagina web.
copyright es una etiqueta que se utiliza para derechos de autor a la pagina para que esta se vuelva irreplicable de manera ilegal.
robots es una etiqueta que se puede combinar con el comando "content=index/noindex" para decidir si se muestra la pagina web durante la busqueta. Para que la pagina se muestre solo se debe de usar "index". si por el contrario quieren que la pagina no se muestre durante la busqueda se debe de usar el comando "noindex".

Etiquetas de Texto

Titulos y subtitulos

Las etiquetas que involucre la letra "h"(h1;h2;h3;h4;h5;h6), normalmnete son utilizadas para la agregacion de titulos y subtitulo de manera descendente junto a los numeros.

Tipo de Letras

la siguentes etiquetas cambian el tipo de letra de un parrafo. "b" es = negrita
"i" es = italica "small" es = letra chica tachado

Para texto

Para generar texto normlmente se utiliza la etiqueta "p" para agregar parrafos, pero despues estan mas opciones como "article" que nos ayudan a agregar un articulo separado, normalmente los articulos son los que contienen al "p"(parrafo)

Separadores

para los separadores vamos a utilizar las etiquetas como:"header", "article", "footer" y "aside". estas etiquetas nos sirven para separar la pagina de manera selectiva y especifica como el encabezado, el articulo en si, el pie de pagina y un cuadro extra de informacion al costado de la pagina.
la etiqueta "header" es utilizada para dividir el encabezado de la pagina, ente suelen ir enlaces, titulo de la pag, logo de la pagina, menus y demas. esta divicion no suele cambiar o moverse a pesar de bajar por la pagina.
la etiqueta "article" es utilizada para agregar informacion en forma de articulo y suele repetirse mucho como division entre noticias. puede contener texto, imagenes, videos, etc, etc.
la etiqueta "aside" normalmente de es utilizada como divisor para agregado de informacion donde el usuario lo ve de manera breve al costado de la pagina de manera rectangular vertical. suele contener texto, imagenes, videos y demas.
la etiqueta "footer" es utilizada para agregar un pie de pagina.

Etiquetas de enlace

las etiquetas de enlaces son utilizadas para redireccionar la vista del espectador hacia otra pagina o a un sitio dentro de esta misma.
la etiqueta para agregar un enlace es "a" aun que esta puede tener diferentes comportamientos dependiendo de la carateristica. naturalmente conlleva la caracteristica "href", que demarca a donde se va a redireccionar(el enlace) y si se quiere que el enlace externo se abra dentro de otra pagina se debe de agregar la caracteristica "target=blank". Pero de esa manera solo de direcciona a un enlace externo, pero para que se redireccione a un sitio dentro de la isma pagina de se le debe agregar "href=#", ademas de que tendremos que agregar la caracteristica "id" para poder identificar a donde se quiere redireccionar.

Para hacer listados

Para hacer listados hay dos tipos de etiquetas. La "ul" y la "ol". la etiqueta "ul" podremos haver una lista normal, pero con la etiqueta "ol" tendremos una una lista numerada ordenada.

Formatos imagen, video y audio

Para agregar formato de de imagen, video o audio tendremos que usar las etiquetas:"img", "video" y "audio". Tendremos que combinarlo con la carateristica "href" que nos indicara la direccion del archivo. Eso sera suficiente para poder agregar un video pero si queremos que el video pueda ser controlado por el usario debemos agregar la caracteristica "controls".

Input

Para lograr un input podremos usar varios tipos de etiquetas que nos serviran para que el usuario puda darnos informacion o iteractuar con la pag.
La etiqueta que vamos a usar es "input", pero lo que va a hacer determinante va hacer la caracteristica. Si usamos la caracteristica "type=text", el usuario podra brindarnos un texto. Si usamos la caracteristica "type=password" el usuario podra darnos una contraseña que saldra en formato de asteriscos. Si utilizamos la caracteristica "type=number" para darle a escoger un numero al usuario. Si colocamos la caracteristica "type=email" nos pertimira darle al usuario agregar un texto en formto de email, denegando cualquier otro formato como invalidos. si utilizamos la caracteristica "type=range" le daremos un rango a escoger al usuario y si le agregamos las caracteristicas "min" y "max" podremos escoger el rango. Con la etiqueta "type=color" podemos darle a escoger al usuario un color. Con la etiqueta "type=date" l usuario podra escoger una fecha. Con la etiqueta "type=time" le daremos al usuario a escoger una hora determinada. Si utilizamos la caracteristica "type=button" agregaremos un boton que el usuario podra pulsar y si lo combinas con la etiqueta "value=" le podemos agregar un nombre.

Input

La etiqueta "selec" nos permite poder darle al usuario varias opciones para seleccionar. Se lo junto con la propiedad "for" que definira la clase de la etique. para poder darle las opciones al usuario debemos poner las etiquetas "option"
